# Brand Guidelines

## Brand Premise

St. Expedite Press is a literary press with a ritual, archival, Southern gothic, and post-institutional identity. The brand should feel deliberate, strange, and editorially serious. It should not feel like a generic startup, a default bookstore template, or a novelty occult theme.

Core tensions:

- Sacred and practical.
- Literary and operational.
- Southern and internet-native.
- Archival and experimental.
- Dark atmosphere and clear action.

## Personality

Use:

- Severe, precise, devotional, marginal, difficult, direct.
- Editorial confidence with minimal explanation.
- Mystery as atmosphere, not obscurity in task flows.

Avoid:

- Cute occult decoration.
- Generic "AI" gradients.
- Overly casual ecommerce copy.
- Institutional nonprofit blandness.
- Decorative density that hides the next action.

## Color System

### Current Core Palette

- Void black: `#050807`
- Signal green: `#2aff8a`
- Relief magenta: `#d96aff`
- Panel black: `rgba(0, 0, 0, 0.76)`
- Strong panel black: `rgba(0, 0, 0, 0.86)`
- Soft panel black: `rgba(0, 0, 0, 0.64)`
- Signal border: `rgba(42, 255, 138, 0.24)`
- Strong signal border: `rgba(42, 255, 138, 0.46)`

### Recommended Role Expansion

- Body text: use a warmer near-ivory tier for long reading, recommended `#e8f8ee`.
- Muted text: use green at reduced opacity for labels, metadata, and helper copy.
- Action text: use signal green for primary actions.
- Warning/accent: use ember orange sparingly for in-progress, attention, or pending states.
- Error: use a red tone only for failures, destructive states, and validation issues.

### Usage Rules

- Signal green should mean brand signal, not every paragraph.
- Magenta should be relief, anomaly, or supernatural heat; do not make it a second default action color.
- Use black surfaces generously, but vary panel strength by mode.
- Background texture belongs in high-atmosphere areas, not inside every dense reading or form surface.

## Intensity Modes

### Ritual

Use for homepage portal, campaign openings, lab instruments, and rare theatrical moments.

Allowed:

- Full green glow.
- Magenta relief flare.
- Twinkle/void backgrounds.
- Large display typography.
- Animated portal/crow imagery.
- Dense atmosphere.

### Editorial

Use for books, about, services, essays, and program explanations.

Allowed:

- Black panels with quieter borders.
- Strong type hierarchy.
- Selective glow around section headers or featured objects.
- More readable body text.
- Wider spacing and calmer rhythm.

### Utility

Use for donate, contact, submit, unsubscribe, errors, confirmations, and settings-like flows.

Allowed:

- Minimal glow.
- Highest text clarity.
- Clear form labels and validation.
- One primary CTA.
- Trust, privacy, and next-step copy.

## Typography

Current fonts:

- Display: `Cinzel`
- Body: `Cormorant Garamond`

Rules:

- Use Cinzel for brand marks, major headers, ritual navigation, and short high-impact phrases.
- Use Cormorant Garamond for editorial prose, descriptions, and body copy.
- Avoid all-uppercase treatment for long text.
- Keep high letterspacing for labels, not paragraphs.
- If a practical utility font is later needed for forms, tables, or code-like metadata, add it deliberately and document it here.

## Imagery

Current image language:

- Crow and glitch portal.
- Void engine/twinkle field.
- Book covers and press objects.
- Minimal line icons and glyph-like symbols.

Rules:

- Use crow/portal imagery as identity anchors, not decorative wallpaper on every card.
- Give books and products clean viewing space.
- Favor high-contrast, low-color imagery that can live in the black/green system.
- Avoid stock photos, generic vector mascots, and smooth corporate 3D imagery.

## Layout And Surfaces

Rules:

- Use large rounded panels for ritual and editorial surfaces.
- Use smaller, calmer radii for dense utility areas.
- Use border and spacing before adding glow.
- Avoid giving every card the same visual weight.
- Let important pages breathe; negative space is part of the brand.

Recommended hierarchy:

- Page frame: atmospheric.
- Main claim: severe and typographic.
- Object or action: clear and close to the user's intent.
- Supporting copy: readable, quiet, and direct.

## Motion

Rules:

- Motion should mark arrival, reveal, loading, or transformation.
- Do not place constant motion next to long reading or form entry.
- Keep hover transitions fast and quiet.
- Provide reduced-motion fallbacks for portal, twinkle, and glow effects.

## Accessibility

Rules:

- Minimum tap target height: 44px.
- Visible keyboard focus on all interactive elements.
- Do not rely on color alone for status.
- Long text should not depend on glow for legibility.
- Validate contrast for green-on-black, muted green, magenta, orange, and red states.

## Do And Do Not

Do:

- Keep the press strange.
- Make task flows calmer than the homepage.
- Use a small number of strong symbols repeatedly.
- Let typography carry hierarchy before decoration.
- Keep source assets in `assets/source/`.

Do not:

- Turn every surface into a portal.
- Introduce a generic sans-serif SaaS look without a clear reason.
- Use purple gradients as a default "modern" style.
- Hide essential actions inside ornate visual treatment.
- Duplicate inline styles when a token or component recipe should exist.
