# Branding Agent Guide

This directory owns brand documentation, design guidance, and exportable tokens.

- Read `../AGENTS.md`, `../ONTOLOGY.md`, and `MEMORY.md` before edits.
- Branding docs do not define runtime behavior unless mirrored into the web app.
- If live CSS tokens change, update `apps/web/public/assets/css/tokens.css` and the relevant branding token docs together.
- Keep design language distinct from RICE even when the publisher relationship is referenced.

Closeout:

- Run `git diff --check` for doc-only changes; escalate if token or web source changes.
- Append a short entry to this directory's `MEMORY.md` and to `../MEMORY.md`.
