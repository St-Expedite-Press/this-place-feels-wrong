# Web JavaScript

Source JavaScript modules for the site shell, page controllers, and shared browser behavior.

Major responsibilities:

- `site-shell.js` for global shell bootstrapping
- `api-client.js` and `form-utils.js` for shared API/form helpers
- page-level controllers such as `books-page.js`, `donate-page.js`, `gallery-page.js`, `contact-page.js`, and `submit-page.js`
- modular Anglossic Compass runtime files prefixed with `lab-anglossic-`
