# Press Web App Agent Guide

This directory owns the Astro public website.

- Read `../../AGENTS.md`, `../../ONTOLOGY.md`, and `MEMORY.md` before edits.
- Do not edit `dist/` by hand; it is generated by `npm run build`.
- For page work, read the target `.astro`, its CSS stack, related JS, and `src/data/site.json` when copy/nav/metadata is involved.
- Use CSS tokens and brand modes from `public/assets/css/tokens.css`; avoid raw colors in component CSS.
- Keep public routes stable unless a breaking change is explicitly requested.

Closeout:

- Run `npm run build` or the broader root `npm run check` for meaningful web changes.
- Append a short entry to this directory's `MEMORY.md` and to `../../MEMORY.md`.
- Update ontology/docs when routes, assets, commands, or ownership change.
