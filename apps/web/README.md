# Web App

Astro static site for St. Expedite Press.

- source: `apps/web/src/`
- authored assets: `apps/web/public/assets/`
- generated output: `apps/web/dist/`
- build command: `npm --prefix apps/web run build`
- preview command: `npm --prefix apps/web run preview`

The homepage and all interior pages are generated from source. Do not edit generated output by hand.
