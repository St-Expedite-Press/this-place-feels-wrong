# Kits Agent Guide

This directory owns reusable scaffolding kits.

- Read `../AGENTS.md`, `../ONTOLOGY.md`, and `MEMORY.md` before edits.
- Keep kit entrypoints, templates, README files, and `agent.config.json` aligned.
- Do not let kit templates silently depend on either RICE or press runtime internals unless explicitly documented.
- Validate referenced paths after template or config changes.

Closeout:

- Update `../ONTOLOGY.md` after kit contract changes.
- Append a short entry to this directory's `MEMORY.md` and to `../MEMORY.md`.
