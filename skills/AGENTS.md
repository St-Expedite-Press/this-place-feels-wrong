# Skills Agent Guide

This directory owns repo-scoped agent skills.

- Read `../AGENTS.md`, `../ONTOLOGY.md`, and `MEMORY.md` before edits.
- Search existing installed, workspace, repo, and curated skills before creating a new skill.
- Prefer updating or consolidating a near-match before adding a competing skill.
- Do not include secrets, `.env` values, private tokens, or unnecessary sensitive context in skills.
- Forward-test materially revised skills when practical.

Closeout:

- Run `npm run check:tooling-integrity` when skill paths or workflow contracts change.
- Append a short entry to this directory's `MEMORY.md` and to `../MEMORY.md`.
